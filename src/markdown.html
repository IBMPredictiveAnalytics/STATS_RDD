STATS RDD Extension Command

Estimate a sharp or fuzzy regression discontinuity design

STATS RDD DEPENDENT=varname* RUNNING=varname*
THRESHOLD=value* TREATMENT=varname INDEP=variables CLUSTER=varname

/OPTIONS KERNEL=TRIANGULAR** or RECTANGULAR or EPANECHNIKIV or QUARTIC or TRIWEIGHT or TRICUBE or GAUSSIAN or COSINE
BANDWIDTH=list of values SETYPE=HC1** or HC3 or CONST or HC or HC0 or HC1 or HC2 or HC4 or HC4m or HC5
MCCRARY=YES** or NO PLOT = YES** or NO

/HELP

* Required
** Default

STATS RDD /HELP. prints this information and does nothing else.

STATS RDD DEPENDENT=y RUNNING=x THRESHOLD=0 INDEP=cov 
/OPTIONS BANDWIDTH=.4 .5 .7 .8 .9.

DEPENDENT names the dependent or outcome variable.

RUNNING names the running or assignment variable. It must have a scale measurement level.

THRESHOLD specifies the cutoff above which the treatment is given.

If TREATMENT is not specified, the design is sharp; if a variable is specified, the design is fuzzy.

INDEP is a list of covariates. They must have a scale measurement level.

CLUSTER specifies a variable whose values are cluster numbers within which the errors are assumed correlated. If specified, robust standard errors are calculated regardless of the SETYPE setting. The R package documentation suggests that if the running variable is discrete that the data be clustered by its values.

KERNEL specifies the kernel to be used.

BANDWIDTH specifies a list of bandwidths. If not specified, the bandwidth is calculated by the Imbens-Kalyanaraman method, and that is used along with half and double the value.

SETYPE specifies the robust standard error method to be used if not overridden by the use of CLUSTER. CONST assumes constant variances while the others assume heteroscedasticity. HC0 is White's estimator. The others are variations of it.

MCCRARY=YES produces the McCrary sorting test

PLOT=YES produces a plot of the outcome vs the running variable and, for fuzzy designs, a plot of the treatment vs the running variable. The plot setting also controls whether the McCrary test produces a plot.
